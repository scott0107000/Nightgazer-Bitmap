# 夜照飛點陣字型系列  Nightgazer-Bitmap
夜照飛 (Nightgazer) 字型系列是修改自螢火飛 (FireFly) bdf 點陣的 ttf 點陣字型。每款字型皆約包含 17,300 個字符，以繁體中文字形為主，支援繁體中文、簡體中文、希臘字符、西里爾字符、日語假名等。

## 簡介
此點陣字型字模來源於 [**螢火飛 bdf 字型**](http://www.study-area.org/apt/firefly-font/bitmaps/)，透過 GPL 授權方式，經格式轉換為 ttf 後，稍修改部份難以辨識之常用符號。

## 鳴謝
在此非常感謝　螢火飛前輩的付出，因原格式文件較不易於現今使用，本人抱持感恩之心完成此次轉換，以冀望更多有需要之人士方便使用。**夜照**，螢火蟲之別名也。希望以此名銘記螢火飛的付出外，也以夜照之名為此字型賦予新生。
